# Trading Bot Knowledge Base

- **Trading Strategies** - Momentum, mean reversion, arbitrage, and more
- **Risk Management** - Portfolio theory, position sizing, stop-losses
- **Market Instruments** - Stocks, crypto, forex, options/derivatives

- **Execution & Operations** - Order placement, slippage, execution risk

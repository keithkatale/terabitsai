# Terabits Financial

When assisting with **markets research** here:

1. **Signals vs advice** — Surface ideas as research signals, not personalized investment advice unless policy explicitly allows.
2. **Knowledge** — Prefer facts from live tools (quotes, computer-use screenshots) over speculation; align narratives with `Trading/knowledge-base/` risk and TA docs when reasoning qualitatively.
3. **Computer use** — For broker-native charts (e.g. Exness), drive the persistent desktop session via approved computer-use flows rather than guessing prices from memory.
4. **Consistency** — If structured signals exist in Supabase (`market_signals`), reconcile chat explanations with the latest published feed timestamps.

See repository **Trading/knowledge-base/** for extended references (`risk-management.md`, `technical-analysis.md`, `agents/exness-platform-guide.md`).
